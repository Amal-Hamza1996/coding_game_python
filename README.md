# Algorithms Challenge – Préparation aux Tests Techniques

Ce repository regroupe **25 exercices d’algorithmique en Python** couvrant les notions essentielles pour réussir les tests techniques en entretien (Data, Backend, Fullstack).  
Les problèmes vont de la **manipulation de chaînes de caractères** aux **structures de données**, en passant par des algorithmes classiques et des exercices de logique.
